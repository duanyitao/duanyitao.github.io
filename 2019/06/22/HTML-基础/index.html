HTML 基本结构

HTML 标签

语法

<标签></标签>

示例

“<html></html>”

规则

  • 用 ‘<’ 和 ‘>’ 括起来
  • 一般成对出现，分为开始标签和结束标签
  • 结束标签比开始标签多了个 ‘/‘
  • 单标签没有结束标签，如 “<hr/>”

HTML 元素

从开始标签 到结束标签 的所有代码，称为 HTML 元素。

HTML 属性

语法

<标签名 属性名1=”值” 属性名2=”值” 属性名3=”值” >……</标签名>

注释

语法：<!– 注释 –>

DOCTYPE

<!DOCTYPE html> 必须放在文档第一行。

网页编码设置

<meta http-equiv=”Content-Type” content=”text/html;charset-utf8”>
